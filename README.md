# Gender Bias Analysis: SHAP vs Attention-based.

This project evaluates gender bias in professional biography classification. It compares SHAP explanations with Attention-based explanations to if language models rely on gendered pronouns to predict occupations

## Prerequisites

Make sure you have python 3.10+ installed. You will need an internet connection to download the model and dataset on the first run.

## Instalations

1. Clone or Download this repository to your device
2. Instal dependencies: Open you terminal/Powershell and run:
"pip install torch numpy matplotlib shap datasets transformers"
